# Documentation

Project documentation and guides.

## Overview

This directory contains the source code for the Viwo documentation site. It uses VitePress to generate a static site from Markdown files, serving as the central knowledge base for the project.

## Contents

- **.vitepress**: VitePress configuration and theme customization.
- **development.md**: Guide for setting up the development environment.
- **index.md**: The landing page of the documentation site.

## Usage

Run the documentation site locally:

```bash
bun run docs:dev
```
