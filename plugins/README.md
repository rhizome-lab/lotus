# Plugins

Optional extensions for the Viwo engine.

## Overview

This directory contains plugins that extend the core functionality of Viwo. Plugins are modular and can be enabled or disabled based on the desired server configuration. They allow for features like AI integration, additional game mechanics, or third-party service connections without bloating the core codebase.

## Contents

- **ai**: Integration with Large Language Models for dynamic content and NPCs.

## Usage

Plugins are typically registered in the server configuration. Refer to each plugin's specific README for installation and configuration details.
