# Seeds

Initial world data.

## Overview

This directory contains the seed data used to populate the world database when the server is initialized. It defines the base objects, verbs, and locations that exist in a fresh Viwo world.

## Contents

- **verbs.ts**: Definitions of core system verbs.
- **objects.ts**: Initial object definitions (e.g., the void, system object).

## Usage

The `seed.ts` script imports these definitions to populate the database.
