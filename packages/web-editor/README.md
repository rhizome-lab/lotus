# Web Editor Package

Web-based code editor components.

## Overview

This package provides the specialized editor components used in the web client and playground for writing and editing ViwoScript. It encapsulates the editor logic and UI.

## Contents

- **src/components**: React components for the editor interface.
- **src/language**: Language service integration and syntax highlighting logic.

## Usage

Import the editor components into a React application to provide a code editing interface.
