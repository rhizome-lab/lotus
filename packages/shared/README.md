# Shared Package

Common utilities and types.

## Overview

This package contains code that is shared across both the backend (core/server) and frontend (web/client) applications. This ensures consistency in type definitions and utility functions throughout the project.

## Contents

- **src/types.ts**: Shared TypeScript interfaces and type definitions.
- **src/utils**: Helper functions used in multiple environments.

## Usage

Import utilities or types from this package to ensure consistency across the codebase.
