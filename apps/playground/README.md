# Playground

Interactive ViwoScript playground.

## Overview

A web-based environment designed for writing, testing, and debugging ViwoScript code in isolation. It provides immediate feedback and is useful for learning the language or testing snippets without a full game server.

## Contents

- **src/App.tsx**: Main application component.
- **src/components**: UI components specific to the playground.
- **src/lib**: Core logic for the playground environment.

## Usage

Start the local development server:

```bash
bun run dev
```
