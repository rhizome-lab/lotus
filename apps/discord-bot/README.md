# Discord Bot

Discord integration for Viwo.

## Overview

This bot bridges the Viwo world with a Discord server. It allows users to interact with the game world, chat with other players, and perform actions directly from a Discord channel.

## Contents

- **src/commands**: Discord slash command definitions.
- **src/events**: Event handlers for Discord gateway events.
- **src/index.ts**: Bot entry point and configuration.

## Usage

Ensure your `.env` file contains a valid Discord bot token.

```bash
bun run start
```
