# CLI

The standard command-line interface for Viwo.

## Overview

This is the primary terminal-based client for users to interact with the Viwo world. It provides a direct text interface for sending commands and receiving output from the engine.

## Contents

- **src/commands**: Definitions of user-facing commands.
- **src/index.ts**: Main entry point that handles input/output loops.

## Usage

Start the interactive CLI session:

```bash
bun run start
```
