# TUI (Terminal User Interface)

A rich terminal-based interface for Viwo.

## Overview

The TUI offers a more visual and interactive experience than the standard CLI, utilizing full-screen terminal capabilities to display game state, menus, and other information in a structured layout.

## Contents

- **src/index.ts**: Application entry point.
- **src/ui**: Components and logic for the terminal user interface.

## Usage

Launch the TUI:

```bash
bun run start
```
