# Admin CLI

CLI tools for system administration and management.

## Overview

This application provides a suite of command-line tools for managing the Viwo instance. It handles tasks such as database migrations, user management, and system configuration, separate from the standard user interface.

## Contents

- **src/commands**: Implementation of administrative commands.
- **src/index.ts**: Entry point for the admin CLI.

## Usage

Run the admin CLI using Bun:

```bash
bun run start [command]
```
