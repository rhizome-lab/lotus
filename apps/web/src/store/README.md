# Store

Global state management.

## Overview

This directory handles the application's global state. It contains the logic for managing user sessions, game state, and other data that needs to be accessible across the application.

## Contents

- **index.ts**: The main store definition and configuration.
- **slices**: Individual state slices for different domains (e.g., auth, chat).

## Usage

Use the provided hooks to access and update the global state from your components.
